cserve

A minimal static file server written in C

What is this?

cserve is a lightweight HTTP static file server written in pure C for Windows. It serves HTML, CSS, and JavaScript files from a directory using a single executable.

Quick start

cserve public

Custom port:

cserve public -p 3000

Folder structure

public/
├── index.html
├── style.css
└── script.js

Features

  • Serves HTML, CSS, and JavaScript
  • No external dependencies
  • Configurable port
  • Native Windows executable
  • Optional GUI wrapper

Security notes

  • Directory traversal (..) is blocked
  • Only static files are served
  • Intended for local development and learning

Status check

If you can read this page, the server is running correctly.

Built with C · Winsock · Win32
